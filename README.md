# Meta-ad-performance-dashboard

• Created an interactive Meta Ad Performance Dashboard using Power BI to track and analyse Facebook and Instagram ad campaigns.
• Analysed key metrics like Impressions, Clicks, CTR, Engagement Rate, Conversion Rate, Purchases and Budget to measure campaign performance.
• Designed dynamic KPI cards to give quick overview of overall ad performance.
• Built visualisations such as bar charts, line charts, donut chart and map to analyse data by age, gender, location, month and ad type.
• Implemented filters and slicers for ad platform and dynamic measure selection to make dashboard user-friendly and interactive.
• Identified trends in weekly and hourly impressions to understand peak performance time.
• Helped in understanding customer behaviour and improving marketing decision making through data insights.
